# CHALLENGE_PANDA
Este es un challenge para la toma de decision de la venta de la tienda que menos lucro produzca.
Este es un challenge cuyo propósito es ayudar al Sr. Juan a determinar cuál de sus tiendas le conviene vender, considerando distintos factores que influyen en el desempeño y la rentabilidad de cada una. Para ello, se analizaron los ingresos totales generados por cada tienda, las categorías de productos más y menos vendidas, las calificaciones promedio otorgadas por los clientes, los productos con mayor y menor demanda, así como el costo promedio de envío asociado a cada tienda. Todo esto el uso de la Liberia pandas y matplotlib


**¿Qué es pandas?**
La biblioteca de software de código abierto Pandas está diseñada específicamente para la manipulación y el análisis de datos en el lenguaje Python. .
Pandas es una biblioteca para Ciencia de Datos de código abierto (open source), construida sobre el lenguaje Python, y que proporciona un enfoque rápido y flexible, con estructuras robustas para trabajar con datos relacionales (o etiquetados), y todo eso de manera sencilla y intuitiva.

**Algunas de sus funciones:**

**Lectura de datos**
pd.read_csv() - Importar archivos CSV.

**Exploración de datos**
df.head(n)- Muestra las primeras *n* filas del DataFrame (por defecto, 5).  
df.tail(n) - Muestra las últimas *n* filas del DataFrame.

**Array de datos**
df.unique()- Devuelve los valores únicos de una columna o serie.

**Agrupación de datos**
df.groupby()- Agrupa los datos según una o más columnas.

**Conteo de valores**
value_counts().idxmax() - Obtiene el nombre del producto más vendido.  
value_counts().max() - Obtiene la cantidad vendida del producto más popular.


**¿Qué es matplotlib?**
Matplotlib es una biblioteca completa para crear visualizaciones estáticas, animadas e interactivas en Python. 
Matplotlib simplifica lo sencillo y hace posible lo complejo.

**Algunas funciones son:**
*   Gráficas de barras (bar)
*   Gráficas de líneas (plot)
*   Diagramas de dispersión (scatter) lista
*   Gráficas de pastel (pie)
*   Histogramas (hist)
